# Criando uma Conta na AWS

## O que é AWS?

A **AWS (Amazon Web Services)** é uma plataforma de serviços de computação em nuvem da Amazon. Com ela, você pode criar servidores, bancos de dados, hospedar sites e muito mais, tudo pela internet.

---

## Por que criar uma conta?

Criar uma conta na AWS permite que você:
- Experimente serviços gratuitos (Free Tier)
- Aprenda sobre computação em nuvem
- Crie projetos reais, como sites, APIs e bancos de dados

---

## Passo a Passo: Criando sua Conta

### 1. Acesse o site da AWS

Vá para: [https://aws.amazon.com/pt/](https://aws.amazon.com/pt/)

### 2. Clique em “Criar uma conta gratuita”

Você verá um botão destacado na página inicial.

### 3. Preencha seus dados

- **E-mail**
- **Senha**
- **Nome da conta**

### 4. Informações de pagamento

A AWS pede um cartão de crédito, mas não cobra nada se você usar apenas os serviços gratuitos.

### 5. Verificação de identidade

Você receberá um código por SMS ou ligação para confirmar sua identidade.

### 6. Escolha o plano

Selecione o plano **Básico (Free)** para começar.

---

## Exemplo Real

Imagine que você quer criar um site para mostrar seu portfólio. Com a conta AWS, você pode:

- Criar uma máquina virtual (EC2) para hospedar seu site
- Usar o S3 para armazenar imagens
- Configurar um banco de dados (RDS) para guardar informações

Tudo isso começa com a criação da conta!

---

## Dica de Segurança

Sempre ative a **autenticação em dois fatores (MFA)** para proteger sua conta.

---

## Resumindo

Criar uma conta na AWS é o primeiro passo para explorar o mundo da computação em nuvem. Experimente, brinque e aprenda!
