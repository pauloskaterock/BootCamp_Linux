# Instalando o Linux Ubuntu em Máquina Virtual (MV)

Instalar o Ubuntu em uma Máquina Virtual (MV) é como criar um "computador dentro do seu computador". Isso permite que você teste, aprenda e experimente o Linux sem alterar seu sistema principal.

## O que é uma Máquina Virtual?

Imagine uma caixa mágica (MV) onde você pode instalar qualquer sistema operacional. Ela roda como um programa comum, mas funciona como um computador completo!

Exemplo: Você usa o Windows, mas quer aprender Linux. Com uma MV, você pode rodar o Ubuntu dentro do Windows, como se fosse um aplicativo.

## Passo a Passo para Instalar o Ubuntu em MV

### 1. Baixe um software de virtualização

- **VirtualBox** (gratuito): [https://www.virtualbox.org/](https://www.virtualbox.org/)
- **VMware Workstation Player** (gratuito para uso pessoal): [https://www.vmware.com/products/workstation-player.html](https://www.vmware.com/products/workstation-player.html)

### 2. Baixe a imagem do Ubuntu

- Acesse: [https://ubuntu.com/download/desktop](https://ubuntu.com/download/desktop)
- Baixe o arquivo `.iso` (imagem do sistema).

### 3. Crie uma nova MV

No VirtualBox, clique em **Novo** e siga as instruções:
- Nome: Ubuntu
- Tipo: Linux
- Versão: Ubuntu (64-bit)
- Memória: Recomenda-se pelo menos 2GB (2048MB)
- Disco rígido: Crie um novo disco virtual (20GB ou mais)

### 4. Instale o Ubuntu

- Inicie a MV recém-criada.
- Quando pedir, selecione o arquivo `.iso` do Ubuntu.
- Siga o assistente de instalação (escolha idioma, usuário, senha, etc).

### 5. Pronto para usar!

Quando terminar, você terá o Ubuntu rodando em uma janela, pronto para explorar, aprender comandos e instalar programas.

## Exemplo Real

> João queria aprender comandos Linux, mas não queria mexer no Windows do seu notebook. Ele instalou o VirtualBox, baixou o Ubuntu, criou uma MV e agora usa o Linux sempre que quer, sem riscos!
